דוגמאות למשחקי AI השראה
1
הגדרות טכניות קריטי
שיקולים

אפשר להוסיף "פרסונה" ל-AI (מורה, מדריך הרפתקאות, מדען) כדי לחזק את הסיפור העוטף.

שיקולים

אם צריך תמונות חיצוניות, ציין שרק קישורי Google Drive מותרים (לא CDN חיצוני).

דוגמת קוד:

// Send when loaded window.parent.postMessage({ type: 'ready', message: 'Loaded' }, '*'); // Send ONLY when game ends function endGame() { const answerObject = { score: totalScore, streak: currentStreak, completed: true, results: allQuestionsResults }; window.parent.postMessage({ type: 'answer', value: answerObject }, '*'); } // Reset function resetGame() { window.parent.postMessage({ type: 'answer', value: {} }, '*'); } // Listen for init (restore state — no postMessage) window.addEventListener('message', (event) => { if (event.data?.type === 'init') { const data = event.data.value; // Restore UI state only } });
מה זה עושה?

חלק זה אחראי על שליחת הציונים והתוצאות מהמשחק חזרה למערכת. הוא שולח את הנתונים רק בסיום המשחק. אין לשנות חלק זה — הוא חיוני לתפקוד התקין של דיווח הציונים.

2
עיצוב כללי + עיצוב דיסציפלינרי עיצוב
שיקולים

אפשר להחליף לצבעים שמתאימים לנושא הדיסציפלינרי — למשל גוונים חמים למדעי הרוח, כחולים למדעים.

שיקולים

אפשר להתאים לגיל היעד — סגנון שטוח ומינימלי לתיכוניסטים, אנימטיבי וצבעוני יותר לחטיבת ביניים.

שיקולים

אם הגובה 700px לא מספיק, שנה ל-auto. אפשר להחליף פונט — Heebo ו-Assistant עובדים היטב בעברית.

שיקולים

אפשר להוסיף טיימר אם הפעילות מוגבלת בזמן, או להסיר "רצף" אם אין מנגנון streak.

ניתן לערוך את הקישורים או להחליף לתמונות אחרות. הפורמט: https://drive.google.com/thumbnail?id=FILE_ID&sz=w1000

שיקולים

אפשר לוותר על אייקונים מותאמים ולהסתפק באייקוני lucide/emoji בלבד — זה מפשט את הפרומפט.

3
סוג הפעילות והתוכן תוכן
שיקולים

אפשר להחליף ל: חדר בריחה, משחק זיכרון, הרפתקת טקסט, או שילוב סוגי שאלות (פתוחות + סגורות).

שיקולים

מומלץ לכלול 8-15 פריטים. אפשר להדביק תוכן גולמי — ה-AI ידע לחלק אותו לשאלות אם תציין "צור שאלות מהתוכן הבא".

הניקוד מצטבר אך ורק עם ההגעה לתשובה הנכונה. תלמיד שטועה יקבל הכוונה ורמז, ויוכל לנסות שוב עד להצלחה ללא איבוד ניקוד בסיסי.
שיקולים

"שליטה" מתאים למבדקים. "מאמץ" מתאים כשרוצים לעודד חקירה ללא חשש מטעויות.

4
מסך סיכום הנתונים פדגוגי
שיקולים

אפשר לבקש שהסיכום יכלול גם "המלצה ללמידה המשכית" — קישור לנושא שהתלמיד התקשה בו.

שיקולים

"שניהם" נותן תמונה מלאה. ציון בלבד מתאים למבחנים, סיכום מילולי מתאים להערכה מעצבת.

שיקולים

אפשר להוסיף מדדים: אחוז שימוש ברמזים, זמן ממוצע לשאלה, או "שאלה שלקחה הכי הרבה זמן".
